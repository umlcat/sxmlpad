readme.txt
==========

The "sxmlpad" software application,
its a Graphical Editor for the "Simplified X.M.L." data format,
a version of X.M.L., without attributes.

(Don't confuse with "Simple XML",
a set of programming libraries for X.M.L.)

This application requires the ukt4laz libraries,
that are not included here, and must be obtained
separatly.

(Also "open/ free source).
